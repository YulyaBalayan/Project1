Методы работы с DOM

Основные методы:

  • document.getElementsByTagName() - получение элементов по тегу
  • document.getElementById() - получение элемента по ID
  • element.innerText - получение/установка текстового содержимого
  • element.innerHTML - получение/установка HTML содержимого
